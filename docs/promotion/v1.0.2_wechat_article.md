# DistroNexus v1.0.2 发布：WSL 管理从未如此简单

大家好！我们非常高兴地宣布 **DistroNexus v1.0.2** 正式发布！🚀

作为一款专注于 Windows Subsystem for Linux (WSL) 管理的现代化 GUI 工具，DistroNexus 始终致力于让您的 Linux 环境构建体验像搭积木一样轻松。

此次更新不仅带来了呼声极高的 **实例管理功能**（移动、重命名等），还首次推出了 **官方安装包**，并为所有 PowerShell 脚本引入了更稳健的日志系统。

---

## 🔥 核心更新亮点

### 1. 高级实例管理：掌控您的 WSL
苦恼于 C 盘空间告急？想要给积灰的实例改个名字？现在这些都可以在 GUI 中一键搞定，无需记忆复杂的命令！

*   **⚡ 启动增强**：
    *   **后台启动**：静默启动实例服务。
    *   **指定目录打开终端**：在设置中配置默认启动路径（如 `~` 或 `/mnt/c/`），直接从 UI 打开终端进入该目录。
*   **🛑 强制停止**：实例卡死或不再需要时，直接在卡片上点击停止。
*   **📦 无损移动 (Move)**：**（重磅功能）** 一键将现有的 WSL 实例迁移到其他磁盘或目录。彻底释放宝贵的系统盘空间！
*   **✏️ 重命名**：随意更改实例的显示名称，管理多环境更清晰。
*   **🔑 凭据管理**：忘记密码或需要重置默认用户？现在的“凭据”按钮可以帮您轻松搞定。

### 2. 包管理器：在线与离线的一站式体验
我们重新设计了包管理页面，使其功能更加全面：

*   **☁️ 在线源同步**：自动从微软官方源获取最新的发行版列表，不再错过任何新版本。
*   **⬇️ 独立下载**：支持单独下载发行版安装包（`.appx`），方便您构建离线安装库或稍后安装。
*   **💾 缓存管理**：直观查看本地缓存占用，一键清理过期文件。

### 3. 全新安装包与数字签名 🛡️
从 v1.0.2 开始，为了提供更安全、便捷的安装体验，我们引入了正规的发布流程：

*   **安装版 (`.exe`)**：
    *   提供标准的 Windows 安装向导。
    *   自动处理依赖配置。
    *   支持卸载和开始菜单快捷方式。
*   **便携版 (`.zip`)**：
    *   依旧为喜爱“绿色软件”的用户保留，解压即用。
*   **数字签名 (Code Signing)**：
    *   感谢 **SignPath.io** 的支持，我们的安装包现已拥有数字签名。
    *   这大大降低了 Windows SmartScreen 的拦截概率，并确保您下载的文件未被篡改，安全可信。

### 4. 稳健性与体验优化
*   **日志系统**：所有后台 PowerShell 操作现已接入统一的日志系统（位于 `logs/` 目录），排查问题有迹可循。
*   **UI/UX 重设计**：
    *   **更现代的布局**：清晰的工具栏与独立的滚动内容区。
    *   **交互式卡片**：直接在实例卡片上集成所有操作按钮，所见即所得。
    *   **人性化设置**：新增“恢复默认设置”及文件夹选择器，配置从此不再繁琐。
