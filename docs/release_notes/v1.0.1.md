# DistroNexus v1.0.1 发布日志

**发布日期**: 2026-01-23

本次更新专注于文档完善、构建流程优化以及用户体验的细节打磨。我们确保了中英文文档的一致性，并为应用添加了精美的图标支持。

## 📝 文档更新 (Documentation)
*   **中英文同步**: 全面更新了 `README_CN.md`，使其与英文文档在功能特性、构建指南和项目结构上保持完全一致。
*   **参数修正**: 修正了命令行参数 `-name` 的描述，明确其含义为“指定实例名称”而非“发行版类型名”。
*   **推广资源**: 新增 `docs/wechat_promotion.md`，为通过社交媒体分享 DistroNexus 提供现成的文章素材。

## 🛠️ 构建与部署 (Build & Deployment)
*   **应用图标**: 实现了图标生成器 (`tools/gen_gear.go`) 和打包逻辑。现在的 Windows 可执行文件 (`DistroNexus.exe`) 将包含内置的齿轮图标。
*   **Fyne CLI 升级**: 更新了 `setup_go_env.sh`，现在会拉取最新版本的 Fyne 命令行工具，确保构建兼容性。
*   **构建脚本优化**: `build.sh` 现在使用 `fyne package` 命令替代原生的 `go build`，以支持资源嵌入。

## 🖥️ 用户体验 (User Experience)
*   **鲁棒性增强**: 在 GUI 中添加了对安装路径（非空目录检查）和实例名称（非法字符检查）的校验逻辑，防止误操作。
*   **界面调整**: 恢复了标准窗口标题栏设计，提高了在不同 Windows版本下的兼容性和稳定性。

## 🧹 其他调整 (Misc)
*   更新了 `.gitignore` 规则，排除 `vendor/`、`build/` 及中间产物，保持代码仓库整洁。
*   标准化了项目目录结构说明。

---
**如何升级：**
请拉取最新代码并重新运行构建脚本：
```bash
./tools/build.sh
```
