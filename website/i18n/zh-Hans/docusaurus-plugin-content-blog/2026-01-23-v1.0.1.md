我们很高兴宣布 **DistroNexus** 的首个版本正式发布！
DistroNexus 是一款功能全面的工具，旨在简化 Windows Subsystem for Linux (WSL) 发行版的管理、下载和自定义安装。它致力于帮助用户在 Windows 上轻松打造专属的 Linux 环境。

<!--truncate-->

## 🚀 核心功能

*   **现代图形界面 (GUI)**：基于 Fyne 构建的跨平台图形化仪表盘，可视化管理一切，无需记忆复杂的命令行参数。
*   **自定义安装路径**：将 WSL 发行版安装到您指定的任意目录或驱动器，不再受限于系统盘默认路径，彻底释放 C 盘空间。
*   **多实例与多版本共存**：轻松安装同一发行版的多个副本（例如 `Ubuntu-Work` 和 `Ubuntu-Personal`），或同时保留不同版本。
*   **离线支持**：自动下载并缓存官方 Appx/AppxBundle 安装包。一次下载，无限次安装。
*   **安全稳健**：内置校验机制，防止意外覆盖现有实例，确保安装过程安全可靠。
*   **自动化支持**：底层的 PowerShell 脚本完全支持无头/静默模式，满足高级用户和 CI/CD 流水线的自动化需求。

## 📦 包含内容

*   **DistroNexus.exe**: 包含内置高清图标的主 GUI 应用程序。
*   **PowerShell 脚本**: 位于 `scripts/` 目录下，负责处理下载和安装的核心逻辑。
*   **配置**: 位于 `config/settings.json` 的简易 JSON 配置文件。
