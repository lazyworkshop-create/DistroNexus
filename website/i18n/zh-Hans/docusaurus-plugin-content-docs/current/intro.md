# 介绍

**DistroNexus** 是一个功能全面的 GUI 应用程序（由 PowerShell 驱动），旨在简化 Windows Subsystem for Linux (WSL) 发行版的管理、下载和自定义安装。

它是你构建 WSL 环境的核心枢纽，让你能够像在工厂中一样定制专属的 Linux 环境。

## 为什么选择 DistroNexus？

Windows Subsystem for Linux (WSL) 是一个强大的工具，但通过命令行管理多个发行版、将它们安装到自定义位置（不仅仅是 `C:` 盘）以及处理离线安装可能会很麻烦。

DistroNexus 通过提供以下功能解决了这个问题：

*   **GUI 便利性**：无需再记忆 `wsl.exe` 的复杂参数。
*   **磁盘空间管理**：轻松将大型发行版安装到辅助驱动器上。
*   **便携性**：在驱动器之间移动实例而不会丢失数据。
*   **离线能力**：一次下载，多次安装，无需重新下载。

## 核心功能

*   **现代图形界面 (GUI)**：基于 Fyne 构建的跨平台图形化仪表盘。
*   **集中下载**：自动下载最新的离线包 (Appx/AppxBundle)。
*   **自定义安装**：将任何 WSL 发行版安装到自定义目录。
*   **高级实例管理**：启动、停止、终止、移动、重命名。
*   **安全检查**：内置验证防止覆盖现有实例。
*   **多版本共存**：轻松安装同一发行版的多个版本。
